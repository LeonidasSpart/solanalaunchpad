# What is ZRP?

ZRP is a **no-code Solana token creation platform** that allows anyone to create, test, and launch SPL tokens on Solana in under 60 seconds — without writing a single line of code.

## Why ZRP?

| Feature | Description |
|---------|-------------|
| **No Code Required** | Create tokens with a simple form, no programming knowledge needed |
| **Free Devnet Testing** | Test your token on Solana Devnet with zero cost |
| **Non-Custodial** | Your keys, your tokens. We never store private keys |
| **Open Source** | Fully auditable codebase on GitHub |

## How It Works

1. **Connect your wallet** (Phantom, Solflare, Backpack)
2. **Fill in token details** (name, symbol, supply, decimals)
3. **Upload a logo** (PNG, JPEG, GIF, WebP)
4. **Choose security settings** (revoke authorities)
5. **Click create** — your token is live in under 60 seconds!

## Key Features

- ✅ 4 templates (Meme Coin, Utility Token, Governance DAO, Simple Token)
- ✅ IPFS metadata storage via Pinata
- ✅ Authority revocation (Mint, Freeze, Update)
- ✅ Share on X (Twitter) integration
- ✅ Token explorer and analytics
