# Sharing Your Token

After creating your token, sharing it is essential for building a community. ZRP makes it easy.

## Share on X (Twitter)

### One-Click Sharing

1. After creating your token, click **"Share on X"**
2. A pre-filled tweet opens:

```

🚀 I just launched $SYMBOL on @ZRP_AI!

Create your own Solana token in 60 seconds → zrp.one

#Solana #SPLToken #Crypto

```

3. Add your own message
4. Click **"Post"**

### Custom Tweets

Try these variations:

**Announcement:**
```

🎉 Excited to announce $SYMBOL is now live on Solana!

Created with @ZRP_AI in under 60 seconds.

Check it out: https://zrp.one
#Solana #Crypto

```

**Community:**
```

🚀 $SYMBOL is here! Join the community and be part of something special.

Built on Solana with @ZRP_AI.

#Solana #Crypto

```

## Share on Telegram

1. Copy your token's mint address
2. Go to your Telegram group
3. Share:
```

🎉 $SYMBOL is live!
Mint Address: [your_mint_address]
View: https://solscan.io/address/[your_mint_address]
Created with @ZRP_AI

```

## Share on Discord

1. Copy your token's mint address
2. Go to your Discord server
3. Share in the appropriate channel

## Share Your Token Link

### Token Details Link

Share the direct link to your token:

```

https://zrp.one/tokens/[your_mint_address]

```

### Solscan Link

Share the Solscan link:

```

https://solscan.io/address/[your_mint_address]

```

## Community Building Tips

| Tip | Why |
|-----|-----|
| **Post regularly** | Keep engagement high |
| **Share updates** | Show progress |
| **Engage with comments** | Build relationships |
| **Create a community** | Telegram or Discord |

## Social Media Strategy

### Before Launch

1. ✅ Create social media accounts
2. ✅ Build a following
3. ✅ Tease the launch
4. ✅ Announce the launch date

### During Launch

1. ✅ Post the announcement
2. ✅ Share the mint address
3. ✅ Engage with responders
4. ✅ Pin important tweets

### After Launch

1. ✅ Post regular updates
2. ✅ Celebrate milestones
3. ✅ Share community posts
4. ✅ Keep engagement high

## What to Share

| Content | Why |
|---------|-----|
| **Token address** | So people can find it |
| **Solscan link** | Verification |
| **ZRP link** | Platform credibility |
| **Logo** | Visual recognition |
| **Roadmap** | Future plans |
