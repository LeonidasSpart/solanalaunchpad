# Connecting Your Wallet

To use ZRP, you need to connect a Solana wallet. This guide shows you how.

## Supported Wallets

| Wallet | Browser Extension | Mobile App |
|--------|-------------------|------------|
| **Phantom** | ✅ Yes | ✅ Yes |
| **Solflare** | ✅ Yes | ✅ Yes |
| **Backpack** | ✅ Yes | ✅ Yes |
| **Ledger** | ✅ Yes | ✅ Yes |

## How to Connect

### Step 1: Install a Wallet

If you don't have a wallet yet:

1. Go to the wallet's website
2. Install the browser extension
3. Create a new wallet
4. Save your seed phrase securely

### Step 2: Go to ZRP

1. Navigate to [zrp.one](https://zrp.one)
2. Look for the **"Select Wallet"** button in the top right

### Step 3: Click "Select Wallet"

1. Click the **"Select Wallet"** button
2. A wallet selection menu will appear

### Step 4: Choose Your Wallet

1. Click on your wallet (Phantom, Solflare, etc.)
2. Your wallet will open

### Step 5: Approve Connection

1. Review the connection request
2. Click **"Connect"** or **"Approve"**
3. You're now connected!

## Connection Status

After connecting, you'll see:

- ✅ Your wallet address shortened
- ✅ Your SOL balance (if available)
- ✅ Network (Devnet/Mainnet)

## Troubleshooting

### "No wallet found"

**Solution:**
1. Make sure your wallet extension is installed
2. Refresh the page
3. Try again

### Connection fails

**Solution:**
1. Make sure your wallet is unlocked
2. Check if you're on the right network
3. Clear browser cache
4. Try a different browser

### Wallet not showing

**Solution:**
1. Check if wallet extension is enabled
2. Click the extension icon
3. Make sure you're logged in

## Disconnecting

To disconnect your wallet:

1. Click your wallet address
2. Select **"Disconnect"** or **"Logout"**
