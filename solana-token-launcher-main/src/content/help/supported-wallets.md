# Supported Wallets

ZRP supports all major Solana wallets. Here's everything you need to know about connecting your wallet.

## Supported Wallets

| Wallet | Type | Browser Extension | Mobile App |
|--------|------|-------------------|------------|
| **Phantom** | 🦊 Popular | ✅ Yes | ✅ Yes |
| **Solflare** | 🔥 Secure | ✅ Yes | ✅ Yes |
| **Backpack** | 🎒 Modern | ✅ Yes | ✅ Yes |
| **Ledger** | 🔐 Hardware | ✅ Yes | ✅ Yes |
| **Torus** | 🔑 Social | ✅ Yes | ✅ Yes |

## How to Connect Your Wallet

### Phantom Wallet

1. Install the Phantom extension from [phantom.app](https://phantom.app)
2. Create a new wallet or import an existing one
3. Navigate to [zrp.one](https://zrp.one)
4. Click **"Select Wallet"** in the top right
5. Select **Phantom** from the list
6. Approve the connection

### Solflare Wallet

1. Install the Solflare extension from [solflare.com](https://solflare.com)
2. Create a new wallet or import an existing one
3. Navigate to [zrp.one](https://zrp.one)
4. Click **"Select Wallet"** in the top right
5. Select **Solflare** from the list
6. Approve the connection

### Backpack Wallet

1. Install the Backpack extension from [backpack.app](https://backpack.app)
2. Create a new wallet or import an existing one
3. Navigate to [zrp.one](https://zrp.one)
4. Click **"Select Wallet"** in the top right
5. Select **Backpack** from the list
6. Approve the connection

## Need SOL?

You'll need a small amount of SOL to cover transaction fees. You can get SOL from:

- [Centralized exchanges](https://www.coinbase.com/) (Coinbase, Binance, Kraken)
- [Solana Faucet](https://faucet.solana.com/) for Devnet testing
- [Jupiter](https://jup.ag/) for swapping

## Troubleshooting

**Wallet not connecting?** Try:
1. Refreshing the page
2. Clearing your browser cache
3. Making sure you're on the correct network (Devnet/Mainnet)
4. Checking if your wallet is unlocked
5. Updating your wallet to the latest version
