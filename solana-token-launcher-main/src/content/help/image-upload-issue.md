# Image Not Uploading

Having trouble uploading your token image? This guide covers common issues and solutions.

## Common Issues

### 1. File Size Too Large

| Issue | Solution |
|-------|----------|
| **File > 5MB** | Compress the image |
| **Large resolution** | Resize to smaller dimensions |

### 2. Wrong File Format

| Issue | Solution |
|-------|----------|
| **Not PNG, JPEG, GIF, WebP** | Convert to supported format |
| **Invalid file** | Check file integrity |

### 3. Network Issues

| Issue | Solution |
|-------|----------|
| **Slow connection** | Wait or try again |
| **RPC error** | Try again later |

### 4. Pinata Issues

| Issue | Solution |
|-------|----------|
| **Pinata API error** | Try again |
| **API key invalid** | Check configuration |

## Supported Formats

| Format | Best For | Max Size |
|--------|----------|----------|
| **PNG** | Logos, transparency | 5MB |
| **JPEG** | Photos, complex images | 5MB |
| **GIF** | Animations | 5MB |
| **WebP** | Modern, efficient | 5MB |

## How to Fix

### Step 1: Check File Size

1. Right-click your file
2. Click **"Properties"** or **"Get Info"**
3. Check the size
4. If > 5MB, compress it

### Step 2: Check File Format

1. Check the file extension:
   - ✅ .png
   - ✅ .jpg or .jpeg
   - ✅ .gif
   - ✅ .webp
2. If not supported, convert it

### Step 3: Compress Image

#### Online Tools

- [TinyPNG](https://tinypng.com)
- [Compressor.io](https://compressor.io)
- [ILoveIMG](https://iloveimg.com)

#### Desktop Tools

- **Photoshop** — Save for Web
- **GIMP** — Free alternative
- **Preview** — Mac built-in

### Step 4: Try Again

1. Refresh the page
2. Re-upload your image
3. Check for success

## Best Practices

### Image Specifications

| Spec | Recommendation |
|------|----------------|
| **Format** | PNG |
| **Resolution** | 512x512 or 256x256 |
| **Size** | Under 2MB for best results |
| **Background** | Transparent |
| **Quality** | High |

### Tips

1. **Square images** display best
2. **High contrast** works well
3. **Simple designs** are recognizable
4. **Transparent background** looks professional

## Error Messages

### "File too large"

```

Error: File too large. Max size is 5MB.

```

**Solution:**
1. Compress the image
2. Resize the image
3. Try again

### "Invalid file type"

```

Error: Invalid file type. Supported: PNG, JPEG, GIF, WebP.

```

**Solution:**
1. Convert the file
2. Use a different format
3. Try again

### "Upload failed"

```

Error: Upload failed. Please try again.

```

**Solution:**
1. Check your connection
2. Try again
3. Contact support if persistent

## Troubleshooting

### File Won't Upload

1. **Try a different browser**
2. **Clear your cache**
3. **Try a different image**
4. **Check internet connection**

### Image Shows After Creation

| Issue | Solution |
|-------|----------|
| **Not showing in wallet** | Wait for IPFS propagation |
| **Not showing on Solscan** | Check metadata URI |
| **Not showing on ZRP** | Refresh the page |
