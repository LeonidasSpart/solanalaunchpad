# Using Templates

ZRP offers four pre-configured templates to help you launch your token quickly with best-practice settings.

## Why Use Templates?

| Benefit | Explanation |
|---------|-------------|
| **Faster Launch** | Settings are pre-optimized |
| **Best Practices** | Industry-standard configurations |
| **Less Research** | No need to study tokenomics |
| **Higher Trust** | Proven settings build confidence |

## Template Comparison

### Meme Coin 🔥

**Best for:** Viral community tokens, memecoins, social experiments

| Parameter | Value |
|-----------|-------|
| Supply | 1,000,000,000 |
| Decimals | 6 |
| Mint Authority | ✅ Revoked |
| Freeze Authority | ✅ Revoked |
| Update Authority | ✅ Revoked |

**Why these settings?**
- Large supply for wide distribution
- 0 decimals for simplicity
- All authorities revoked for maximum trust

## How to Choose

| If You Want To... | Choose |
|-------------------|--------|
| Create a viral memecoin | Meme Coin |
| Build a serious project | Utility Token |
| Start a DAO | Governance DAO |
| Test or experiment | Simple Token |

## Customization

All templates can be customized before creating:

1. **Change supply** — Adjust to your needs
2. **Change decimals** — Any value between 0-9
3. **Toggle authorities** — Revoke or keep any authority

## Start from Scratch

If none of the templates fit your needs:

1. Click **"Start from Scratch — Full Customization"**
2. Set all parameters manually
3. Complete control over every setting
