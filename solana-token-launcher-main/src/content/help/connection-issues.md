# Troubleshooting Connection Issues

Having trouble connecting your wallet to ZRP? Here's how to fix common issues.

## Common Connection Problems

| Problem | Likely Cause | Solution |
|---------|--------------|----------|
| **Wallet not showing** | Extension not detected | Refresh page |
| **Connection fails** | Wallet locked | Unlock wallet |
| **Network mismatch** | Wrong Solana network | Switch network |
| **RPC error** | Connection timeout | Check RPC settings |

## Quick Fixes

### 1. Refresh the Page

Sometimes the simplest fix works best:

1. Press `Ctrl+R` or `Cmd+R`
2. Wait for page to reload
3. Try connecting again

### 2. Unlock Your Wallet

1. Click the wallet extension icon
2. Enter your password
3. Try connecting again

### 3. Check Your Network

1. Open your wallet
2. Check the network (Devnet/Mainnet)
3. Match the network on ZRP

### 4. Clear Browser Cache

1. Go to browser settings
2. Clear cache and cookies
3. Restart browser
4. Try again

### 5. Try a Different Browser

| Browser | Support |
|---------|---------|
| **Chrome** | ✅ Best |
| **Brave** | ✅ Good |
| **Firefox** | ✅ Good |
| **Safari** | ⚠️ Limited |

## Wallet-Specific Fixes

### Phantom

1. **Check extension** — Is it up to date?
2. **Try incognito** — Open Chrome incognito
3. **Disable other extensions** — They may conflict
4. **Reinstall** — Uninstall then reinstall

### Solflare

1. **Check settings** — Ensure Solflare is active
2. **Browser compatibility** — Try Chrome
3. **Clear cache** — Clear browser cache
4. **Use the app** — Try the mobile app

### Backpack

1. **Check connection** — Is Backpack connected to Solana?
2. **Update app** — Make sure you have the latest version
3. **Reconnect** — Try reconnecting

## Error Messages

### "No wallet found"

```

No wallet found. Please install Phantom, Solflare, or Backpack.

```

**Solution:** Install a wallet extension.

### "Connection failed"

```

Failed to connect to wallet. Please try again.

```

**Solution:** Check your wallet and try again.

### "Network mismatch"

```

Network mismatch. Please switch to Devnet/Mainnet.

```

**Solution:** Switch your wallet to the correct network.

### "User rejected"

```

Transaction rejected by user.

```

**Solution:** Approve the transaction in your wallet.
