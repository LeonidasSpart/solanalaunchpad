# Reporting Suspicious Activity

If you encounter suspicious activity on ZRP, here's how to report it.

## What to Report

### Suspicious Tokens

| Type | Description |
|------|-------------|
| **Scam tokens** | Tokens designed to steal funds |
| **Impersonation** | Tokens pretending to be legitimate |
| **Fake projects** | Projects with no real value |
| **Pump and dump** | Schemes to manipulate price |

### Suspicious Behavior

| Behavior | Description |
|----------|-------------|
| **Phishing** | Attempting to steal private keys |
| **Spam** | Unsolicited promotional messages |
| **Harassment** | Abusive or threatening behavior |
| **Impersonation** | Pretending to be someone else |

### Technical Issues

| Issue | Description |
|-------|-------------|
| **Security vulnerability** | Bug in the platform |
| **Exploit** | Something being exploited |
| **Data breach** | Potential data exposure |

## How to Report

### Via Website

1. Go to the [Contact page](/contact)
2. Fill in the form
3. Include:
   - Description of the issue
   - Relevant links or screenshots
   - Any additional information

### Via Email

1. Email: `support@zrp.one`
2. Subject line: `Security Report: [Brief Description]`
3. Include all relevant details

### Via X (Twitter)

1. Tag @ZRP_AI
2. Share a brief description
3. Don't share sensitive information publicly

## What to Include

### For Suspicious Tokens

| Detail | Why |
|--------|-----|
| **Token address** | To identify the token |
| **Transaction ID** | If relevant |
| **Screenshots** | Visual evidence |
| **Description** | What happened |

### For Suspicious Users

| Detail | Why |
|--------|-----|
| **Username/handle** | To identify the user |
| **Chat logs** | Evidence of behavior |
| **Transaction history** | If relevant |
| **Description** | What happened |

### For Technical Issues

| Detail | Why |
|--------|-----|
| **Steps to reproduce** | How to replicate |
| **Error messages** | What you saw |
| **Screenshots** | Visual evidence |
| **Browser/device** | Environment details |

## What Happens Next

### Process

1. ✅ Report is received
2. ✅ Team investigates
3. ✅ Action is taken if needed
4. ✅ You may be contacted for more info

### Actions

| Action | Description |
|--------|-------------|
| **Removal** | Suspicious content removed |
| **Flagging** | Token flagged as suspicious |
| **Warning** | User receives a warning |
| **Ban** | User banned from platform |
| **Investigation** | Further investigation needed |

## Protecting Yourself

### Before Reporting

1. **Document everything**
2. **Don't engage** with suspicious actors
3. **Don't share private keys**
4. **Don't send money**

### After Reporting

1. **Stay vigilant**
2. **Monitor your accounts**
3. **Update your passwords**
4. **Share information** with the community

## Emergency Situations

### If You've Been Scammed

1. **Stop** all transactions
2. **Document** everything
3. **Report** to ZRP
4. **Report** to relevant authorities
5. **Warn** the community

### If Your Wallet Is Compromised

1. **Immediately** move funds to a new wallet
2. **Create** a new wallet
3. **Revoke** all approvals
4. **Report** to ZRP

## Community Responsibility

### How You Can Help

1. **Report suspicious activity**
2. **Warn others** in the community
3. **Share security tips**
4. **Stay informed**

### What to Share

- ✅ Verified security alerts
- ✅ Safety tips
- ✅ Best practices
- ❌ Unverified rumors
- ❌ Panic-inducing messages
