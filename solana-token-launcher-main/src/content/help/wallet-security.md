# Wallet Security Best Practices

Your wallet is the key to your crypto assets. Keeping it secure is essential.

## Never Share Your Seed Phrase

| Do | Don't |
|----|-------|
| ✅ Write it down and store securely | ❌ Share it with anyone |
| ✅ Use a hardware wallet | ❌ Take a photo of it |
| ✅ Store multiple copies | ❌ Store it online |

> ⚠️ **Warning:** ZRP will never ask for your seed phrase. Anyone who does is a scammer.

## Use Hardware Wallets

### Why Hardware Wallets Are Best

| Feature | Benefit |
|---------|---------|
| **Offline Storage** | Keys never touch the internet |
| **Physical Approval** | Must confirm on device |
| **Protected** | Immune to malware |

### Recommended Hardware Wallets

- **Ledger** — Most popular, supports Solana
- **Trezor** — Supports Solana
- **NGRAVE** — Premium option

## Strong Passwords

### Best Practices

| Practice | Example |
|----------|---------|
| **Long passwords** | At least 12 characters |
| **Complex** | Upper, lower, numbers, symbols |
| **Unique** | Different from other accounts |
| **No reuse** | Never reuse passwords |

### Password Manager

Use a password manager like:
- 1Password
- Bitwarden
- LastPass

## Two-Factor Authentication (2FA)

### Recommended 2FA Methods

| Method | Security Level |
|--------|----------------|
| **Authenticator App** | High |
| **SMS** | Low (avoid) |
| **Email** | Low (avoid) |

### Authenticator Apps

- Google Authenticator
- Authy
- Microsoft Authenticator

## Avoid Phishing

### Red Flags

| Sign | What It Means |
|------|---------------|
| **Unexpected emails** | Could be phishing |
| **Suspicious links** | Hover to check URL |
| **Urgent requests** | Scammers create urgency |
| **Typos** | Look for spelling mistakes |

### How to Stay Safe

1. **Always check URLs** — `zrp.one` not `zrp-one.com`
2. **Bookmark important sites** — Use bookmarks
3. **Never click links** — Type URLs manually
4. **Check the lock icon** — Site should be secure

## Secure Your Device

| Practice | Why |
|----------|-----|
| **Keep software updated** | Security patches |
| **Use antivirus** | Detect malware |
| **Avoid public Wi-Fi** | Potential for attacks |
| **Lock your screen** | Physical security |

## What ZRP Does

### We Never:

- ❌ Request your seed phrase
- ❌ Store your private keys
- ❌ Control your funds
- ❌ Ask for personal information

### We Always:

- ✅ Sign transactions locally
- ✅ Keep you in control
- ✅ Use industry standards
- ✅ Provide security guides

## What to Do If Compromised

1. **Immediately** move funds to a new wallet
2. **Create** a new seed phrase
3. **Revoke** any approvals
4. **Contact** support if needed
