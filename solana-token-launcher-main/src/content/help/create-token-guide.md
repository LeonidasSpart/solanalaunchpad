# How to Create a Solana Token

Creating a Solana token on ZRP is simple and takes less than 60 seconds. Here's everything you need to know.

## Prerequisites

Before you start, make sure you have:

1. **A Solana wallet** (Phantom, Solflare, or Backpack)
2. **Some SOL** in your wallet for transaction fees
3. **A token idea** (name, symbol, supply, decimals)

## Step 1: Connect Your Wallet

1. Go to [zrp.one/create-mint](https://zrp.one/create-mint)
2. Click **"Select Wallet"** in the top right
3. Choose your wallet (Phantom, Solflare, etc.)
4. Approve the connection in your wallet

## Step 2: Choose a Template (Optional)

You can start from scratch or use one of our templates:

| Template | Best For |
|----------|----------|
| **Meme Coin** | Viral community tokens |
| **Utility Token** | Governance & utility |
| **Governance DAO** | Voting and delegation |
| **Simple Token** | Testing and private use |

## Step 3: Fill Token Details

| Field | Description |
|-------|-------------|
| **Name** | Full token name (e.g., "Pepe Coin") |
| **Symbol** | Ticker symbol (e.g., "PEPE") |
| **Description** | Brief description of your token |
| **Total Supply** | Total number of tokens |
| **Decimals** | 0-9 (0 for memecoins, 9 for standard) |

## Step 4: Upload Image

1. Click **"Choose File"**
2. Select your token logo (PNG, JPEG, GIF, WebP)
3. Max size: 5MB

## Step 5: Configure Security

Choose which authorities to revoke:

| Authority | What It Does |
|-----------|--------------|
| **Mint Authority** | Prevents minting more tokens |
| **Freeze Authority** | Prevents freezing accounts |
| **Update Authority** | Makes metadata immutable |

## Step 6: Create Token

1. Review your token details
2. Click **"Create & Mint Token"**
3. Approve the transaction in your wallet
4. Wait for confirmation

## Step 7: Share Your Token

Once created, you can:

- ✅ Share on X (Twitter)
- ✅ View on Solscan
- ✅ Copy mint address
- ✅ Create another token

---

## Troubleshooting

**Transaction failed?** Make sure you have enough SOL for fees.

**Token not showing?** Manually add it to your wallet using the mint address.

**Image not uploading?** Check file size and format (max 5MB, PNG/JPEG/GIF/WebP).
