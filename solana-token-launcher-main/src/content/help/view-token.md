# Viewing Your Token

After creating a token, you can view it in multiple places. This guide shows you how.

## On ZRP

### Token Explorer

1. Go to the [Token Explorer](/tokens)
2. Find your token in the list
3. Click on it to view details

### Token Details Page

The Token Details page shows:

| Info | What It Shows |
|------|---------------|
| **Name & Symbol** | Your token's identity |
| **Mint Address** | Unique identifier |
| **Total Supply** | How many tokens exist |
| **Decimals** | How divisible it is |
| **Creator** | Who created it |
| **Created Date** | When it was created |
| **Security Status** | Which authorities are revoked |
| **Your Balance** | How many you hold |

### Dashboard

1. Click **"Dashboard"** in the header
2. You'll see all your tokens
3. Each token shows:
   - Name and symbol
   - Balance
   - Actions (view, burn, etc.)

## On Solscan

### View on Solscan

1. Copy your token's mint address
2. Go to [Solscan.io](https://solscan.io)
3. Paste the mint address in the search bar
4. Press Enter

### What You'll See on Solscan

| Section | Info |
|---------|------|
| **Overview** | Name, symbol, supply, decimals |
| **Holders** | Who holds the token |
| **Transactions** | All activity |
| **Metadata** | Your token details |
| **Authority Info** | Which authorities are revoked |

## In Your Wallet

### Phantom

1. Open Phantom
2. Click **"Tokens"**
3. Your token should appear in the list
4. If not, click **"Add Token"** and paste the mint address

### Solflare

1. Open Solflare
2. Click **"Assets"**
3. Your token should be visible
4. If not, click **"Add Token"** and paste the mint address

### Backpack

1. Open Backpack
2. Click **"Tokens"**
3. Your token should appear
4. If not, click **"Add Token"** and paste the mint address

## Common Issues

### Token Not Showing in Wallet

**Solution:**
1. Click **"Add Token"** or **"Import Token"**
2. Paste the mint address
3. Confirm

### Wrong Balance Displayed

**Solution:**
1. Refresh the page
2. Wait for the network to update
3. Check Solscan for accurate balance

### Token Not Found in Explorer

**Solution:**
1. Make sure you're on the right network
2. Check the mint address for typos
3. Wait a few minutes for the network to update
